# Quickstart

OpenLIT Ops Cloud is a SaaS workspace for LLM observability dashboard.

## Fast Path

1. Open OpenLIT Ops Cloud and choose a bounded pilot workflow.
2. Enter public-safe examples for LLM observability dashboard.
3. Review the generated queue, evidence, or approval state with the owner.
4. Use pricing and checkout links when the pilot needs a paid workspace.
5. Export the resulting report or evidence for the team record.
